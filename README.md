

# StockSocial — DB Frontend

Frontend interface for our Social Stock Trading Platform database project.

## How to Run

### 1. Clone the repository
git clone https://github.com/Ebaed135/stock-frontend.git

### 2. Open in VS Code
- Open VS Code
- File → Open Folder → select the cloned `stock-frontend` folder

### 3. Install Live Server (one time only)
- Click the Extensions icon on the left sidebar (or press `Ctrl+Shift+X`)
- Search **Live Server** (by Ritwick Dey)
- Click Install

### 4. Run the app
- Right-click `stock_frontend.html` in the file explorer
- Click **Open with Live Server**
- The app will open in your browser at `localhost:5500`
